## Build Setup
```bash
1. Clone repository
$ git clone https://github.com/arwysyah/otomo
```
2. Install depedencies

```bash
# with npm
$ npm install

# or with yarn
$ yarn install
```

3. Setup your environment variable in `.env` files (if not exists, create your own).

```env
DB_HOST      = 'localhost'
DB_USER      = 'root'
DB_PASSWORD  = ''
DB_DATABASE  = 'otomo'
```

4. Start API server

```bash
$ npm run server
```
